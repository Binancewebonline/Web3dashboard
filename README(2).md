---

## 🚀 How to Publish Your Own

1. Upload `index.html` to your GitHub repo
2. Go to **Settings → Pages**
3. Select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Click **Save**

Your public site will appear at:  
`https://<your-username>.github.io/<your-repo-name>`
